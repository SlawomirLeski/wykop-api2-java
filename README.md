## How to use it

1) Please update file [wykop.properties](src/main/resources/wykop.properties) - add your credentials

2) From command line: 
```
mvn clean install exec:java -Dexec.mainClass="pl.wykop.App" -Dexec.cleanupDaemonThreads=false
```
